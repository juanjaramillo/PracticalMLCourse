Introduction

Using devices such as Jawbone Up, Nike FuelBand, and Fitbit it is now possible to collect a large amount of data about personal activity relatively inexpensively. These type of devices are part of the quantified self movement ??? a group of enthusiasts who take measurements about themselves regularly to improve their health, to find patterns in their behavior, or because they are tech geeks. One thing that people regularly do is quantify how much of a particular activity they do, but they rarely quantify how well they do it. In this project, your goal will be to use data from accelerometers on the belt, forearm, arm, and dumbell of 6 participants.

The goal of this project is to predict the exercise that a set of people (20) did based on the measurements present in the dataset.

We have a data frame with 19622 observations with at most 160 variables most of them numeric. Five different kinds of exercises were being made by different people:

unique(training$classe)

Given the different measurements we are going to try to predict which classes correspond to some new observations. To make it easier, we will try to get only the numeric variables from the training database so that we can have a consistent data frame for predictive purposes.

nv <- vector()
for(i in 1:dim(training)[2]-1)
{
  if(class(training[,i])=="numeric")
  { 
    nv <- c(nv,i)
  }
}
training2 <- training[,c(nv,dim(training)[2])]

We try to eliminate the NAs from the dataframe with 0s so that we dont have problems for classification, even though the data may seem sparse.

training2[is.na(training2)] <- 0.0
